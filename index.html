¿Que es Plc Node?

PlcNode es un módulo análogo a al funcionamiento de un controlador lógico programable que se considera como un nodo donde se concentra información, se controla y optimiza las acciones de un proceso industrial por medio de la implementación de internet y está orientado al procesamiento de entradas y salidas donde en las entradas se utilizan sensores y en las salidas se utilizan actuadores.

¿Demanda de Plc´s con internet industrial?


A nivel global, los controladores lógicos programables son demandados, debido a que se necesitan procesos automatizados para eficientar la productividad y generación de productos en masa, con la llegada de cuarta revolución industrial cada vez la industria se encuentra con mayor competencia entre sí y con ello la demanda de estos productos Plc´s de ultima generación con internet son demandados para la implementación de concepto de internet de las cosas en el área industrial.

Node-Red supports

Node-RED es una herramienta de desarrollo basada en flujo para programación visual desarrollada originalmente por IBM para conectar dispositivos de hardware, API y servicios en línea como parte del Internet de las cosas. Node-RED proporciona un editor de flujo basado en navegador web, que se puede utilizar para crear funciones de JavaScript. Los elementos de las aplicaciones se pueden guardar o compartir para su reutilización.
